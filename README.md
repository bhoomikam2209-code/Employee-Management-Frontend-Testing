# Employee-Management-Frontend-Testing
Employee Management System with Frontend Manual Testing using Qase



## 📌 Project Overview
This project demonstrates frontend manual testing of a simple Employee Management System developed using HTML, CSS, and JavaScript. The application includes a Login page, Dashboard, Add Employee page, and Employee List page.

Manual testing was performed on the Login module using **Qase Test Management** to verify the application's functionality.

---

## 🎯 Objectives
- Develop a basic web application.
- Perform frontend manual testing.
- Create and execute test cases using Qase.
- Document the testing process.

## 🧪 Testing Performed

### Frontend Testing
Verified that the user interface behaves correctly based on user interactions.

### Functional Testing
Verified that the Login functionality works as expected.

### UI Testing
Checked labels, buttons, input fields, and page navigation.

### Negative Testing
Verified appropriate error messages for invalid login attempts.
